FOLDER ICONS INFORMATION

This folder contains custom icon images for the desktop folders and system elements.
Place your icon images here with the following names:

- 2d_art_icon.png - For the 2D Arts folder
- 3d_art_icon.png - For the 3D Works folder
- pixel_icon.png - For the Pixel Arts folder
- animation_icon.png - For the Animations folder
- tattoo_icon.png - For the Tattoos folder
- lily_icon.png - For the About Me folder
- windows.png - For the Windows Start button in the taskbar

The recommended image sizes are:
- Folder icons: 64x64 pixels for best display
- Windows icon: 24x24 pixels for the Start button

Image formats supported: PNG, JPG, GIF.

If icon files are missing, the application will display default styled icons.
